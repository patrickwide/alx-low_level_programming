## C - Functions for Printing Numbers and Shapes

This is a list of C functions and programs for printing numbers and shapes:

- `print_numbers`: This function prints the numbers 0 through 9, each followed by a newline character.
- `print_most_numbers`: This function prints the numbers 0 through 9, each followed by a newline character, except for 2 and 4.
- `more_numbers`: This function prints the numbers 0 through 14, ten times, each followed by a newline character.
- `print_line`: This function takes an integer argument and prints a horizontal line of that length using the '-' character, followed by a newline character.
- `print_diagonal`: This function takes an integer argument and prints a diagonal line of that length using the '\' character, followed by a newline character.
- `print_square`: This function takes an integer argument and prints a square of that size using the '#' character, followed by a newline character.
- `fizzbuzz`: This program prints the numbers from 1 to 100, each followed by a newline character. For multiples of three, it prints "Fizz" instead of the number. For multiples of five, it prints "Buzz". For numbers that are multiples of both three and five, it prints "FizzBuzz".
- `print_triangle`: This function takes an integer argument and prints a right-angled triangle using the '#' character, followed by a newline character.
- `largest_prime_factor`: This program finds and prints the largest prime factor of the number 612852475143, followed by a newline character.
- `print_number`: This function takes an integer argument and prints it, followed by a newline character.
